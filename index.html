Our robot made it to the quarterfinals of every tournament we participated in.

Friends Find Me
2015

Created the Android part of a multi-platform application. The application uses gps to determine where nearby friends are and displays an arrow on your smartwatch pointing to the nearest one. This part of the application was developed using the Android SDK, Java, and XML. This application was developed at BoilerMake Hackathon at Purdue. This was the first full Android application I've worked on, so I learned a lot over one weekend. Also, this was the first application I've worked on that had to communicate with the Pebble Smartwatch as well as the backend server.
