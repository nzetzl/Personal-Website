Robotics Club
2012

Created and debugged a program in a language based on C that controlled a robot in a dynamic, live, and competitive environment. I was the sole programmer for a team of 8 students in my high school robotics team. The robot had to move autonomously for 30 seconds, accomplishing a variety of tasks to score points for the team. Then, the robot moved via remote controllers for the next minute and a half. I was in charge of programming the robot for both modes and making sure everything went as planned. Our robot made it to the quarterfinals of every tournament we participated in.

Friends Find Me
2015

Created the Android part of a multi-platform application. The application uses gps to determine where nearby friends are and displays an arrow on your smartwatch pointing to the nearest one. This part of the application was developed using the Android SDK, Java, and XML. This application was developed at BoilerMake Hackathon at Purdue. This was the first full Android application I've worked on, so I learned a lot over one weekend. Also, this was the first application I've worked on that had to communicate with the Pebble Smartwatch as well as the backend server.
